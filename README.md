# Meu Linktree | front-end

## Sobre

Linktree é uma plataforma de social-linking e redes sociais que opera num modelo freemium, seu objetivo é fornecer uma página única (ou mesmo landing page) onde quaisquer pessoas ou empresas consigam disponibilizar os seus diversos links, em um mesmo lugar.

Além de permitir a inclusão de links para todas as suas redes sociais, o Linktree também permite aos seus usuários algumas opções de personalização simples, como, adicionar um avatar e escolher um dos temas disponíveis para sua página. 

Com uma assinatura Pro o Linktree oferece mais benefícios aos seus usuarios, como, mais opções de personalização, análises mais detalhadas, integração para inscrição de e-mail, remoção de logotipo da Linktree, etc.
